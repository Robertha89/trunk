25-Feb-2014

Carnie updated:

  • Most of the core-systems in the game are being reworked in an attempt to reduce lag. Don't get too excited, there isn't much in the way of speed-improvments in this update.
    Please report any irregularities/bugs/freakish-events HERE. Thankyou!
    PS a general note about power-systems: the power subsystem runs less frequently now, so when you're turning on power at an APC, do not be alarmed if it takes a few seconds to power up. This can be tweaked easily, so please let me know if it feels wrong or whatever.
  • You can now set the levels at which the different power channels of APCs turn off in AUTO mode. Before, these values were hardcoded. For example, this would allow engineers to configure the engineering APC to prioritise equipment power over lighting and environmental systems, meaning the singulo containment would be the last thing to fail.
    The way this works is pretty simple (although it doesn't look simple). The percentage value you see when you have access to an APC, indicates the minimum amount of charge the APCs cell must have for that system to be on. Obviously, if you set the minimum to 0%, that system will always attempt to be on - This is referred to as AlwaysOn. On the otherhand, if set to 100%, it will always be off (since the cell never has more than 100% charge) - This is referred to as AlwaysOff. Anything in-between is referred to as Auto.
    There are buttons for These settings, and a 'Default' button to reset this percentage to its default value. This would be the same as clicking 'Auto' in the old system.
  • Solars have been reworked somewhat. The solar control must be synced with the solars and tracker manually. This is simply to reduce the amount of legwork the system has to do every tick. As before, solars and tracker must be wired before they can be discovered by the solar-control computer. Solar arrays should generally generate about 8000 Watts-per-second when they are directly facing the sun. Solars now rotate instantly.
  • The sun's behavior is slightly more varied in the speeds it can rotate at
  • Admins: The MC stats are now much more accurate, so don't be alarmed by those big numbers, those systems were always that slow ,_, I've identified the problems causing them and they're on my TODO list.
